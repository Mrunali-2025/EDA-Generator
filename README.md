# 📊 AI/ML Datasets EDA Generator

A modern, web-based **Exploratory Data Analysis (EDA)** tool built using **Streamlit** and **ydata-profiling**.
It helps users transform raw CSV data into detailed insights with just a few clicks.

---

## ✨ Features

### 🎨 Modern UI/UX Design

* Neumorphic design with clean card-based layout
* Beautiful gradient theme for better visual experience
* Easy-to-use interface for beginners

### 📂 Upload & Analyze

* Upload any CSV dataset
* Automatic data processing and analysis
* Handles missing values and data types

### 📊 Automated EDA Report

* Generates a complete profiling report using **ydata-profiling**
* Includes:

  * Data summary
  * Correlation analysis
  * Missing values visualization
  * Distribution plots

### ⚡ Fast & Efficient

* One-click report generation
* Saves time compared to manual EDA

---

## 🛠️ Tech Stack

* **Frontend & App**: Streamlit
* **EDA Library**: ydata-profiling
* **Language**: Python
* **Libraries**: Pandas, NumPy, Matplotlib, Seaborn

---

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/REPO_NAME.git
cd REPO_NAME
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the app

```bash
streamlit run app.py
```

4. Open in browser

```
http://localhost:8501
```
